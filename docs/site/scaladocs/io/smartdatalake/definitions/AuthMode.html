AuthMode 

sealed trait AuthMode extends AnyRef

Authentication modes define how an application authenticates itself to a given data object/connection

You need to define one of the AuthModes (subclasses) as type, i.e.

authMode {
  type = BasicAuthMode
  user = myUser
  password = myPassword
}
